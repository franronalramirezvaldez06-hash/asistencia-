#  Sistema de Gestión de Asistencia Fullstack

Sistema profesional diseñado para el control de asistencia académica, utilizando una arquitectura desacoplada y moderna.

## 🛠️ Tecnologías Utilizadas

### **Backend (Java)**
* **Framework:** Spring Boot 3
* **Persistencia:** Spring Data JPA + MySQL
* **Seguridad:** Configuración de CORS personalizada para integración con Angular.

### **Frontend (Angular)**
* **Versión:** Angular 17+
* **Estado:** Uso de **Signals** para una reactividad ultra rápida.
* **Diseño:** UI Moderna con soporte para Modo Oscuro (Dark Mode).

## 📋 Características Principales
* Gestión completa de Alumnos (CRUD).
* Filtrado dinámico por grupos académicos (A, B, C, D).
* Registro de estados de asistencia en tiempo real.
* Interfaz responsiva y optimizada para el usuario.
